Fee Structure 2-20 Rule

Example Calculation

Let's consider an initial investment of 2.5 lakhs (INR) with a quarterly return of almost 12%:

  • Asset Management Fee (0.5%): 2,50,000 = 1250 INR
  • Threshold Value (2% of Initial Investment): 2,50,000 * 0.02 = 5,000 INR
  • Remaining Profit after Threshold: 30,000 - 5,000 = 25,000 INR
  • Profit Sharing Fee (20%): 25,000 * 0.2 = 5,000 INR
  • Total Profit After Fees: 20,000 + 5,000 - 1250 = 23,750 INR

The threshold value represents the minimum level at which we apply profit charges, allowing fees to be incurred only once the fund surpasses this specified threshold.

If the profits exceed 15%, the profit-sharing fee will be 45% for every rupee above the 15% High-water mark.
